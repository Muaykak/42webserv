3. Data Table

Testing repetitive tags and tabular data formatting.

ID Status Code Message Description
1 200 OK The request succeeded normally.
2 201 Created The request succeeded, and a new resource was created as a result.
3 400 Bad Request The server could not understand the request due to invalid syntax.
4 403 Forbidden The client does not have access rights to the content.
5 404 Not Found The server can not find the requested resource.
6 405 Method Not Allowed The request method is known by the server but is not supported by the target resource.
7 413 Payload Too Large Request entity is larger than limits defined by server.
8 500 Internal Server Error The server has encountered a situation it does not know how to handle.
9 501 Not Implemented The request method is not supported by the server and cannot be handled.
10 505 HTTP Version Not Supported The HTTP version used in the request is not supported by the server.

5. Code Blocks

Testing preformatted text tags preserving whitespace.


int main(int argc, char **argv) {
    if (argc != 2) {
        std::cerr << "Usage: ./webserv [config_file]" << std::endl;
        return (1);
    }
    
    try {
        WebServ server(argv[1]);
        server.run();
    } catch (std::exception &e) {
        std::cerr << "Error: " << e.what() << std::endl;
    }
    
    return (0);
}
